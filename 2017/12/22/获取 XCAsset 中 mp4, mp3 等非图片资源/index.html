Posted on 2017-12-22 | In 开发学习 | Views:

场景

在 xcasset 添加 Data Set 内容, 如 .mp3, .mp4 等非图片内容时, 通过 NSBundle 无法获取文件路径, -[NSBundle pathForResource:ofType:] 方法返回为 nil.

目标

不改变文件存放位置的前提下, 获取文件的路径或二进制内容.

前置
